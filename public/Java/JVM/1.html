# 类加载(ClassLoader)

  • 在Java代码中,类型的加载连接与初始化都是在程序运行期间完成的
  • Java虚拟机(JVM即一个进程)与程序的生命周期
  • 在如下几种情况下结束JVM
    • 程序执行正常结束
    • 执行了System.exit()
    • 程序在执行过程中遇到了异常或错误而异常终止
    • 由于操作系统出现错误而导致JVM进程终止

# 三个过程(加载,连接,初始化)

  • 加载: 查找并加载类的二进制数据
  • 连接:
    • 验证: 确保被加载的类的正确性
    • 准备: 为类的静态变量分配内存,并将其初始化为默认值
    • 解析: 把类中的符号引用转换为直接引用
  • 初始化: 为类的静态变量赋予正确的初始值
