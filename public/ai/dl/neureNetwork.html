# 神经网络

# 1. 生物神经元

以下对生物神经元的描述并非完全,仅做参考

# 结构

  • 细胞体: 神经元主体,由细胞核、细胞质、细胞膜等组成,细胞膜对细胞液中的不同离子 的 通透性不同,使得产生离子浓度差,从而出现内负外正的静息电位
  • 树突: 通过树突接收来自其他神经元的输入信号
  • 轴突: 传出细胞体产生的输出电化学信号
  • 突触: 神经元间通过一个的轴突末梢和其他神经元的细胞体或者树突进行通信连接,相当于接口

# 信息处理机制

  • 信息产生: 在某一给定时刻,神经元总是处于静息、兴奋和抑制三种状态之一。在外界的刺激下,当神经元的兴奋程度大于某个阈电位时,神经元被激发而发出神经脉冲.
  • 传递与接收: 神经脉冲信号沿轴突传向其末端的各个分支,通过突触完成传递与接收。突触有兴奋性突触和抑制性突触两种,当兴奋性突触的点位超过某个阈电位时,后一个神经元就有神经脉冲输出,从而把前一个神经元的信息传递给了后一个神经元。
  • 信息整合: 接收各个轴突传来的脉冲输入,根据输入可到达神经元的不同部位,输入部位不同,对神经元影响的权重也不同。在同一时刻产生的刺激所引起的电位变化大致等于各单独刺激引起的电位变化的代数和神经元对空间和时间上对输入进行积累和整合加工,从而决定输出的时机和强弱.

# 2. M-P模型(麦卡洛克-皮特斯模型)

       1943年心理学家McCulloch和数学家W·Pitts基础生物神经元的基础特点抽象与提出了M-P模型,这个模型对生物神经元信息处理过程进行了简化和概括.

四个特性:

  • 多输入单输出
  • 不同输入权重不同
  • 多输入累加整合
  • 阈值特性

# 多层神经元诞生的意义

在单层神经元时代,因为一层的神经元无法解决异或问题

异或我们都知道,相同为0,不同为1

       那么什么是异或问题呢?

  • 男和女才能生小孩
  • 找两张图片间的不同
  • 异或来压缩影音,将连续的部分帧中相同的部分只存一次
  • 寻找超级远外的哪些星发生了移动

       如抽象人类看静止或者运动缓慢的事物不会特别在意,但对一只向你俯冲而来的老鹰却格外在意.是否可以理解为异或差距很大.

       另外也可以通过异或来判断两个id间的距离.异或得到的值越小,这两个ID越接近。

ID1：01000
ID2：01010
异或值：00010表示ID1和ID2比较相近

为了解决这个问题,多层神经网络便诞生了

可以通过多个单层组合成异或神经网络

# 3. 多层感知器中的激活函数

注意区分激活函数和损失函数        生物的神经元一层一层连接起来,当神经信号达到某一个条件(阈值),这个神经元就会激活,然后继续传递信息下去

       为了使用神经网络来解决非线性可分性的问题,采取神经网络的输入端和输出端插入更多的神经元.

       但是注意,如果插入的部分都是简单的ax+b,那么就会产生一种问题就是无论插入多少层,最终结果都是线性的:

例:

y = ax+b x = cz+d y = acz+(ad+b) - 依然是线性的

故引入激活函数

# 常见的激活函数

# relu

       当x小于0,则信号被屏蔽掉,x大于0则原样输出,比较常用

# sigmoid

       范围在[-1,1]间,一般是用来算概率的,但是 - 因为sigmoid在值极大或极小时会产生梯度几乎为0的情况,所以sigmoid并不常用于中间层

# tanh

# Leak relu

常用在生成网络中,如gan

# 6. 多分类问题 - softmax激活函数与softmax交叉熵损失函数

sigmoid解决的是二分类问题,输出一个概率值

那么如何解决多分类问题呢? 一个思路是运用多次二分类 但有一个比较直接的模型 - softmax模型

       神经网络的原始输出不是一个概率值，实质上只是输入的数值做了复杂的加权和与非线性处理之后的一个值而已，那么如何将这个输出变为概率分布？

Softmax模型

       输入是向量,其功能是将数值概率化,最终结果是在所有的标签中找一个概率最大的结果.那个结果对应的标签就是答案. 且 softmax各样本分量之和为 1,当只有两个类别时，与对数几率回归完全相同

       对于softmax的激活函数,有两种比较常用的损失函数,即softmax交叉熵

在tf.keras里，对于多分类问题我们使用

  • categorical_crossentropy
  • sparse_categorical_crossentropy 来计算softmax交叉熵

其中第一个是用在one-hot编码 第二个则是用在输出标签的情况下

使用tf.keras将顺序编码转成one-hot tf.keras.utils.to_categorical()

# 7. CNN - 卷积神经网络

       卷积主要概念分为卷积核,padding,移动步长,filter.

# 卷积核(kernal)

kernal即卷积核,一个卷积核相当于一个矩阵,通过卷积核来将原矩阵信息提取,简化.

# 通道(channel)

通道即一个图片会被分为几个通道,如RGB则为三通道,RGBA可分为四通道

# filter

可以理解为立体卷积,如一个立体物体可以在三个方向上移动.普通RNN可以认为是1D卷积,2D图像CNN可以认为是2D卷积,对立体物体的check可以认为是3D卷积. 一般称为 Conv2D - Conv3D

# 怎么实现一个图片最终序列化成一维向量?

对于2D卷积,一层多个卷积核,每个卷积核将原图片信息提取到较小的矩阵中.

对于每个图片下一层继续多个卷积核.

直到最后一层的卷积核使得最终结果为1*1的矩阵.结合起来就是向量.

# 池化层是什么?

池化层其实就是一个和卷积核比较相似的东西,不过池化层要比卷积核的复杂度低得多,它的作用主要使用来扩大视野,简化计算. 比如最大池化,平均池化,就是直接在一个范围内圈定最大值,平均值,作为提取出来的值.

stride: n - 即每个步伐移动n步.

全局平均池化

Flatten层是将二维数据展开成一维. GAP - GlobalAveragePooling2D 也是将其转为一维,计算全局平均值转为一维.

如下图

在图像处理时,随着层数的增加,fliter的增加,最终会有一个很厚的数据集,如何将这些数据集引入到Dense(即普通全连接层)进行计算,GAP就担任了这样的一个角色.

# 激活函数与bias

激活函数和bias(偏置变量)是直接作用在每一层数据上的.

如Relu(conv2D层),则是对该层的节点上二维数据中每一个数据都进行Relu

一般卷积核我们选择3*3 为了防止过拟合,我们可以适当添加Dropout

# 批标准化

标准化即归一化,批标准化是在每一层隐含层后面都进行标准化.

用来解决梯度消失和梯度爆炸 使用批标准化之后可以使用更深的网络

一般批标准化放在激活函数前,层之后
