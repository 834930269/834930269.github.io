# L系统

L系统是通过L文法来确定递归方式的分形法

一个L系统主要包含3个元素

  • 字母表: L系统的字母表由系统可能出现的合法字符组成,比如"ABC",意思是L系统中的任何"语句"只能包含这3个字符。
  • 公理: 公理是一个语句,用于描述系统的初始状态,举个栗子,对于字母表为"ABC"的L系统,他的公理可以是"AAA" "B" 或"ACBAB"
  • 规则: L系统的规则首先作用于公理,然后递归作用于结果,每次作用都会产生一个新的语句.一个规则包括"前身"和"继任者"两部分。比如,规则"A->AB"的意思就是,将所有的"A"替换成"AB".

# 最简单的L系统: Lindenmayer模拟藻类生长

字母表: A B
公理: A
规则: (A -> AB) (B -> A)

# 那么L系统和绘图有什么关系呢?

事实上,我们可以将L系统的语句当做绘图的指令,如:

字母表: A B
公理: A
规则: (A -> ABA) (B -> BBB)

我们可以用以下方式翻译L系统语句:

  • A: 向前画一个线段
  • B: 向前移动,不画任何线段

上面所示的L系统可视化后即为康托尔集

# 常用的L系统字母表: Tutle graphics

"FG+=[]" 含义如下:

  • F: 画一个线段,然后向前移动
  • G: 向前移动(不画任何线段)
  • +: 向右转
  • -: 向左转
  • [: 保存当前位置
  • ]: 恢复之前位置

该字母表被称为海龟绘图法

为啥叫海龟绘图法

电脑屏幕中有一只海龟,你可以对它下一些命令: 向左转,向左转,画一个线段等,在这个文法下就可以实现这种功能

对字母表的翻译:

  • F: line(0,0,0,len); translate(0,len);
  • G: translate(0,len)
  • +: rotate(angle)
  • -: rotate(-angle);
  • [: pushMatrix();
  • ]: popMatrix();
函数介绍

translate(x,y): 把新的原点设置在了（x,y）;

rotate(PI/4): 把图形旋转了pi/4;

pushmatrix(): 功能保存当前坐标系统堆栈;

popmatrix(): 恢复之前的坐标系统;

resetMatrix(): 复原坐标系统

停止处理从连续执行draw（）中的代码。 如果调用了loop（），则draw（）中的代码将再次开始运行。 如果在setup（）中使用noLoop（），它应该是块内的最后一行。

当使用noLoop（）时，不可能在诸如mousePressed（）或keyPressed（）之类的事件处理函数中操作或访问屏幕。 相反，使用这些函数调用redraw（）或loop（），它将运行draw（），它可以正确地更新屏幕。 这意味着当没有调用noLoop（）时，不会出现任何图形，并且可能不使用像saveFrame（）或loadPixels（）这样的函数.

请注意，如果草图被调整大小，即使在指定了noLoop（）之后，也将调用redraw（）来更新草图。 否则，草图将进入奇数状态，直到调用loop（）。

# 复集 生成代码

对于原Processing代码只允许同时绘制一个L文法系统.

对架构进行合理的调整,可以在同一张画布上画上多个L文法树

LSystem代码 (解析文法,生成下一个文法) 
class LSystem {

  String sentence;     // 句子
  Rule[] ruleset;      // 带入Rule对象数组
  int generation;      // 统计迭代

  public LSystem(String axiom, Rule[] r) {
    sentence = axiom;
    ruleset = r;
    generation = 0;
  }

  //L系统规则
  void generate() {

    StringBuffer nextgen = new StringBuffer();
    for (int i = 0; i < sentence.length(); i++) {
      char curr = sentence.charAt(i);
      String replace = "" + curr;
      for (int j = 0; j < ruleset.length; j++) {
        char a = ruleset[j].getA();
        if (a == curr) {
          replace = ruleset[j].getB();
          break; 
        }
      }
      nextgen.append(replace);
    }
    sentence = nextgen.toString();

    generation++;
  }

  String getSentence() {
    return sentence; 
  }

  int getGeneration() {
    return generation; 
  }

}

Turtle (根据文法绘制) 

class Turtle {

  String todo;
  float len;
  float theta;
  public int count=0;
  int limited = 2;
  public LSystem lsys; 
  
  Turtle(float l, float t,int lim,Rule[] rule,String r) {
    len = l; 
    theta = t;
    count=0;
    limited = lim;
    lsys=new LSystem(r, rule);
    todo = lsys.getSentence();
  } 

  void render() {
    stroke(0,175);
    for (int i = 0; i < todo.length(); i++) {
      char c = todo.charAt(i);
      if (c == 'F' || c == 'G') {
        line(0,0,len,0);
        translate(len,0);
      } 
      else if (c == '+') {
        rotate(theta);
      } 
      else if (c == '-') {
        rotate(-theta);
      } 
      else if (c == '[') {
        pushMatrix();
      } 
      else if (c == ']') {
        popMatrix();
      }
    } 
  }

  void setLen(float l) {
    len = l; 
  } 

  void changeLen(float percent) {
    len *= percent; 
  }

  void setToDo(String s) {
    todo = s; 
  }

}

Turtle t1,t2,t3,t4;
Rule[] ruleset;
void setup() {
  size(800, 800);
  ruleset = new Rule[4];
  ruleset[0] = new Rule('F', "FF+[+F-F-F]-[-F+F+F]");
  ruleset[1] = new Rule('S', "FF+[+F-F-F]-[+F-F-F]");
  ruleset[2] = new Rule('D', "F+F+F+F+F+F+F+F");
  ruleset[3] = new Rule('Z', "FFF+[FF--]FF-FF");
  t1 = new Turtle(30, radians(25),3,ruleset,"F");
  t2 = new Turtle(30, radians(25),3,ruleset,"D");
  t3 = new Turtle(40, radians(20),3,ruleset,"S");
  t4 = new Turtle(100, radians(1000),3,ruleset,"Z");
  smooth();
}

void draw() {
  background(255);  
  fill(0);

  draw3(300,height/2,2,t1);
  draw3(350,height/2,2,t2);
  draw3(500,height/3,1.85,t3);
  draw3(200,height/3,1.85,t4);
  //noLoop();
}

void draw3(int x,int y,float delta,Turtle t){
  resetMatrix();
  translate(x,y);
  rotate(-PI/delta);
  t.render();
  while(t.count<t.limited){
    println(t.count);
    t.count++;
    pushMatrix();
    t.lsys.generate();
    t.setToDo(t.lsys.getSentence());
    t.changeLen(0.5);
    popMatrix();
    //redraw();
  }
}
