# leetcode 6

# 6

Z 字形变换

题目

将一个给定字符串根据给定的行数，以从上往下、从左到右进行 Z 字形排列。

比如输入字符串为 "LEETCODEISHIRING" 行数为 3 时，排列如下：

L  C   I   R  
E T O E S I I G  
E   D   H   N
1
2
3

之后，你的输出需要从左往右逐行读取，产生出一个新的字符串，比 如："LCIRETOESIIGEDHN"。

请你实现这个将字符串进行指定行数变换的函数：

string convert(string s, int numRows);

# 解题思路

先想到取余,然后会发现一个周期是numRows*2-2

第一行首字符下标是0,然后是0+numRows*2-2... 第二行首字符下标是1,然后第二个下标是numRows×2-2-1,然后这两个下标依次向后加numRows×2-2 ... 最后一行首字符下标是numRows-1,然后依次加...

# AC代码

#include<bits/stdc++.h>
using namespace std;

string convert(string s, int numRows) {
    if(s=="") return "";
    if(numRows==1) return s;
    string ans="";
    //观察会发现就是取余逐渐向中间靠拢,且n*2-2一定是偶数
    //第一行是0,第二行是余数等于1和numRows*2-3
    //第三行是2和numRows*2-4
    for(int i=0;i<numRows;++i){
        if(i==0 || i==numRows-1){
            int first = i;
            int id=first;
            while(id<=s.length()-1){
                //printf("%c",s[id]);
                ans+=s[id];
                id+=numRows*2-2;
            }
        }else{
            int first = i;
            int second = numRows*2-2-i;
            int id1=first,id2=second;
            while(id1<=s.length()-1){
                //printf("%c",s[id1]);
                ans+=s[id1];
                id1+=numRows*2-2;
                if(id2<=s.length()-1){
                    //printf("%c",s[id2]);
                    ans+=s[id2];
                    id2+=numRows*2-2;
                }
            }
        }
    }
    return ans;
}
int main(){
    string str="PAYPALISHIRING";

    cout<<convert(str,4);
    return 0;
}
