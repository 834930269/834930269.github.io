# leetcode 3

# 3

Longest Substring Without Repeating Characters

题目

Given a string, find the length of the longest substring without repeating characters.

# 解题思路

暴力O(n2n^{2}n2) 肯定TLE

       首先考虑DP,即:

  • dp[i]代表包含第i个位置时最大无重复字串的长度
  • dp[i-1]包含第i-1位置上---
  • dp[i] = 如果在dp[i-1]中存在和str(i)相等的字符,且该字符在dp[i-1]中位于第j个位置,则dp[i]=i-j,否则dp[i]=dp[i-1]+1

那么问题就是如何快速查找是否有相等的字符,如果尝试暴力后搜,那么和顺序暴力没任何差别,考虑字符映射,一共只有24个字母.

但当我们考虑到字符只有26个时,我们可以将暴力变成24*n,试试. 然后第一波被干掉了,为什么呢?因为它的字符集是整个ASCII字符集,(虽然想到了...)然后把映射表改成256,然后运行到985/9..挂了,看了下,是因为他的字符串长度为3W,而我申请的只有1000的长度.然后尝试改成4W,也挂了,估计是因为申请内存的时间过长?考虑不使用申请空间.

过了,靠,也算不上是DP,就是优化,但是绝对不是最好的结果.

因为从前置状态转移到后置状态时的状态信息是记录着的,如果优化,完全可以从这方面上剪掉一大堆的枝

复杂度: 256*n

# 考虑优化

dp[i]记录的是以i结尾的最长无重复子串 那么对于dp[i-1]而言,可以直接从dp[i]的长度-1开始向前搜索,而不需要每次都搜索256次.

前置条件,dp[i]肯定是搜索到有重复的字符串了,这个字符串不一定是str(i),dp[i-1]的最后一位是str(i-1)

但是考虑到既然状态是可以继承的,那么是否可以直接用两个浮标代替这个区间?,然后不断地左移右区间浮标,同时左移左区间浮标.这之间的长度最长的就是最后的结果,复杂度应该是O(n)乘上一个常数.

# AC代码

时间: 13.0%

class Solution {
public:
    int temp[257]={0};
    int max_v=0,cnt=0;
    int lengthOfLongestSubstring(string s) {
        if(s=="") return 0; 
        for(int i=s.length()-1;i>=0;--i){
            cnt=0;memset(temp,0,sizeof(temp));
            for(int j=0;j<256 && i-j>=0;++j){
                if(temp[s[i-j]]==0){
                    cnt++;
                    temp[s[i-j]]=1;
                    max_v=max(max_v,cnt);
                }else break;
            }
        }
        return max_v;
    }
};

# 优化后代码

时间: 73.83%

class Solution {
public:
    int max_v=0,j;
    int temp[256]={0};
    int lengthOfLongestSubstring(string s) {
        if(s=="") return 0; 
        if(s.length()==1) return 1;
        j=s.length()-2;
        temp[s[s.length()-1]]=1;
        max_v=1;
        for(int i=s.length()-1;i>=0;--i){
            if(max_v==256) break;
            if(i!=s.length()-1){
                temp[s[i+1]]=0;
                temp[s[i]]=1;
                if(i==j) j=i-1;
            }
            for(;j>=0;j--){
                if(temp[s[j]]==0){
                    temp[s[j]]=1;
                    max_v=max(max_v,i-j+1);
                }else{
                    break;
                }
            }
        }
        return max_v;
    }
};
