# leetcode 5

# 5

最长回文子串

题目

给定一个字符串 s，找到 s 中最长的回文子串。你可以假设 s 的最大长度为 1000。

# 解题思路

暴力O(n3n^{3}n3) 肯定TLE

方式,枚举区间,然后遍历判断是否是回文

考虑怎么优化,枚举区间时肯定会出现无数的无效区间,即该区间肯定不是回文区间,那么怎么剪掉这部分的枝?

考虑枚举中心点,向两边扩散,遇到不满足回文的时候即停止.

分两种情况

  1. 回文字符个数为奇数个,枚举中心点,向两边扩散
  2. 回文字符个数为偶数个,枚举左侧中心点,如果右侧相等继续向两边扩散,因为上一层循环包含了下一层的向右判断是否是回文(有点绕,但意思就是只需要check一边就可以了)

# AC代码

#include<bits/stdc++.h>
using namespace std;

int max_l=0,max_r=0,max_len=0;
string longestPalindrome(string s) {
    if(s.length()<=1)return s;
    for(int i=0;i<s.length();++i){
        //奇数
        int ct=1,j;
        for(j=i-1;j>=0 && i+i-j<=s.length()-1;--j){
            if(s[j]==s[i+i-j]) ct+=2;
            else break;
        }
        if(ct>max_len){
            max_len=ct;
            max_l=j;max_r=i+i-j;
        }
            //printf("%d %d\n",max_l,max_r);
        //偶数,右扩散
        j=i+1;ct=0;
        while(j<=s.length()-1 && i+i-j+1>=0){
            if(s[j]==s[i+i-j+1]){
                ct+=2;
                j++;
            }else break;
            if(ct>max_len){
                max_len=ct;
                max_l=i+i-j+1;
                max_r=j;
            }
        }
        //printf("%d %d\n",max_l,max_r);
    }
    //printf("%d %d\n",max_l,max_r);
    return s.substr(max_l+1,max_r-max_l-1);
}

int main(){
    string str="ccc";

    cout<<longestPalindrome(str);
    return 0;
}
