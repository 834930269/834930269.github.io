# 在废土中读书

有人曾说过我的许多人生路径将不可能一帆风顺,我承认了,但也有人告诉过我.路是人走出来的,你没有走过,为什么别人就一定会失败呢?(from 算法启迪的老师-板蓝根),大学我按照非既定的道路走到了最后,耐人寻味的同学和老师也在路上慢慢的转变了对我的态度.
可能后面就开始膨胀了,想要按照大学的道路继续走下去,但始终会发现,孤独的标签一直存在.是不是自己的问题依然存在呢?
或许独自会走得很快,或许一个人会无牵无挂，没有任何责任的牵绊.可这终究不是任何人想要的.
废土之上有许多人同样经历过白眼与挫折,连续经历十几年挫折与白眼的人也不占少数.又有多少人能走出来呢?
你曾想依靠技术与实力以及做的事情与别人交流,而后来你会发现,这一切只会带来别人对你的尊重,而不是友善的快乐.
或许你有许多同行的人愿意主动的留在你的身边,但是你会发现,那些早就经历过社会毒打多年的老朋友会希望你能够与他同行.那些依然简单的孩子也需要得到你的帮助.
但这些关系都并非对等的.即尊重与诚信以及不(愿意)谙世事的附属品罢了.

那就请至少在

废土

中读书,直到你真正的快乐,并且带着不长的风尘.

文中不乏言辞犀利的批判与自我批判,不掩盖伤口才能对症下药,更好地愈合是贯穿主线的思维方式.

# Note.1 被讨厌的勇气

# [日] 岸见一郎 古贺史健

# 序

张沛超教授是职业心理咨询师,他在序中介绍自己30岁之前沉迷于弗洛伊德式的精神分析,教授认为过分的认同并忠诚于一种信条,就会不知不觉间开始通过一根管子去观察世界和人生,甚至这根管字可能更细.但当译者读到了阿德勒的书籍时,他体味到了阿德勒被动方文华影响甚多,并且阿德勒对于当下的重视,对于人际的理解,对于勇气和决定的重要性的再三确认都是教授深感敬佩与沉醉的.

# 序二 《自我的枷锁和解放》

知乎专栏作家 动机在杭州

序者自述曾在懵懂少年时敏感自卑如本书的主人公,觉得人生一片灰暗(这点倒是不同于他),有一天序者在书店偶遇阿德勒的《超越自卑》，忽然觉得自己还能抢救一下(好,好,好).
序者言: 我们很多的心理困扰都来自于社会和他人的期待和评价,正是这种评价体系,造成了人类自身的骄傲和自卑,而人类又经常借"爱"之名,行支配与控制只实.在阿德勒的眼中,理想的人际关系大概是"我爱你,但与你无关".他认为每个人的课题都是分离又独特的.我有自己的课题,至于你愿不愿意与我共同研究,那是你的课题.

# 序三 《人唯有在能够感觉自己有价值时，才有勇气》

胡慎之

序者讨厌心灵鸡汤,序者讲述了一个故事,一个要强的40岁职业女性,从28岁开始,就患有严重的洁癖,她一直怀疑丈夫可能有外遇,经常对孩子控不住的怒火,此外,对以往的闺蜜也甚少邀约,因她碍于自己的大姐大身份,无法去和他人分享自己的感觉,怕被别人耻笑,所以她感觉自己很孤独.

但她在生活中是一个对所有人都非常好的人,甚至能够把生活中所有的事情都打理的有板有眼,她是个"好人".有强烈的同情心,但同时也"好人没好报"的抱怨(这点我不认同),他会抱怨被人欺骗,被朋友利用,甚至在遭遇感情背叛时也会产生一些不好的想法.

讨论到自我价值时,她总说:"感觉自己一直不如别人,害怕自己对别人没用".在她的人际关系中,似乎总是有着某种纵向的规律,我要高于你,或你高于我,没有一种关系体验是"我不出众,但很平等."的合作式关系.

序者言: 如果能早点读这本书,就可以终结她用"牺牲自己,讨好他人"获得价值感的病态模式,体会到"共同体感受".

# 引言

哲人贯穿引言所述为一:
世界是简单的,关键是你看待世界的方式.

一个小故事:
或许你知道,井水的温度是恒定的,常年在18度左右。这是一个客观的数字,无论谁测都一样。但是,夏天喝到的井水感觉凉爽,而冬天饮用时就感觉温润,温度恒定在18度,但夏天与冬天饮用的感觉却大不相同.

这并不是错觉,而是不同环境下的主观.是否想要摘掉这个主观的墨镜去正视这个世界,就是最需要的

勇气.

之后会用极简的话语陈述每一章的内容

# 第一夜

Section.1(Page 5 to 7) - 再怎么"找原因",也没法改变一个人
问: 一个人认为自己有心灵创伤而不想出门.
答: 阿德勒考虑的不是过去的原因,而是现在的目的.即他并非是因为不安的情绪不想出门,而是由于不想到外面去,所以才制造出不安的情绪
同样说: 这个人感受到的困扰是真实的,但这些困扰都是为了达到"不出去"这个目的而制造出来的.
阿德勒强调我们如果移植依赖于原因论,最终就会永远止步不前.

Section.2(Page 8 to 10) - 心理创伤并不存在

  • 阿德勒心理学明确否定心理创伤
  • 决定我们自身的不是过去的经历,而是我们自己赋予过去经历的意义
  • 如果一个人一直闭门不出,他可能并非是害怕外面的世界,而是想要得到父母的关注
  • 我们大家都是为了某种目的而活着

tag: 你之所以对于老师的任务极度关注,对于任务极度苛刻,其原因自当不是你充满了责任心,而应当是你因为某种原因必须去完成.只有完成这些苛刻的任务,你才可能得到想要的结果.
