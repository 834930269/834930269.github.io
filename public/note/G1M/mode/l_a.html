# 范德蒙行列式

范德蒙行列式
1 x0 ... x0^n
1 x1 ... x1^n

...

 ... ... ...
1 xn   xn^n

如上表所示,主要用来表示多项式

例: f(x) = 1 + a1x + a2x^2+...+anx^n

范德蒙行列式解唯一存在的方式是 : 其行列式值不等于0 -> 每个x都不相等

|A| = (xi-xj)[任意i,j的乘积] ≠ 0
