# 滤波器

# 均值滤波器

3*3的滤波器,卷积方式进行,均值滤波器意在将滤波器范围内的值统一成一样的,即平均化.

同样也有带权重的均值滤波器.即并非是3*3里全是1. 如:

	  1 1 1

1/9 * 1 1 1

	  1 1 1

和

	   1 2 1

1/16 * 2 4 2

	   1 2 1

       其中f为输入图像,w为滤波淹模,g为滤波结果,其他参数,filtering_mode是相关滤波,还是卷积,boundary_options是边界填充方式,p代表填0,还有三个参数不做概述,size_optionsfull表示图像变胖,即不处理,same表示图像和原图像大小保持不变.

# 注意

       空域滤波前必须对图像进行im2double操作,因为计算并非是整数计算,会带有小数,而省去小数会造成误差.

f = imread('a.jpg')
f1 = rgb2gray(f)
f2 = im2double(f1)
w = ones(31);
gd = imfilter(f,w);
imshow(gd,[])

# 典型空域滤波器

即工具箱中预定义的滤波器 w = fspecial('type',parameters)

type为筛选器类型,parameters为进一步指定滤波器的参数

  • 'average' 3*3均值滤波
  • 'disk'
  • 'laplacian' 拉普拉斯滤波
  • 'gaussian' 高斯滤波
  • 'log' 先高斯,再拉普拉斯

# 锐化空间过滤器

个人感觉锐化的思路就是利用灰度间细微的差异扩大化

# 什么是锐化

  • 锐化的目的是突出图像的细节
  • 通常是通过空间查分来完成的
  • 数字函数的导数是用差分来定义的
  • 对于一阶导数,我们要求他必须满足
    • 平面段必须为零(灰度值恒定的区域)
    • 在灰度阶跃或斜坡开始时必须非零
    • 斜坡上一定是非零
  • 对于二阶导数,我们要求他必须满足
    • 在平坦段(灰度值恒定的区域)中必须为零
    • 在灰度级阶跃或斜坡开始和结束时,必须是非零的
    • 在坡度恒定的斜坡上必须为零.

# 差分

       一维函数一阶导数的一个基本定义就是差分.

f对x的偏导=f(x+1)-f(x)

       二阶导数:

对x二阶偏导=f(x+1)+f(x-1)-2f(x)

# 拉普拉斯算子

f对x二阶偏导+f对y二阶偏导=[f(x+1,y) + f(x-1,y) + f(x,y+1) + f(x,y-1)] - 4f(x,y)

# matlab实现

f = imread('a.jpg')
f1= rgb2gray(f)
w4 = fspecial('laplacian',0)
w8 = [1 1 1;1 -8 1;1 1 1]; #自己构建的滤波器
f1 = im2double(f1)
g4 = f1-imfilter(f1,w4,'replicate')
g8 = f1-imfilter(f1,w8,'replicate')
imshow(f)
figure.imshow(g4)
figure.imshow(g8)

# 中值滤波 - 常用

gm = medfilt2(fn);
imshow(gm)

#2
gms = medfilt2(fn,'symmetric');
