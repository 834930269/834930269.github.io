# 变换

不同于操作一个函数生成一个新的值 而是操作一个新的函数生成一个新的函数

# 数学表示

\begin{equation} g(x,y) = T[f(x,y)] \end{equation}

       f(x,y): 输入图像        g(x,y): 输出图像        T: 在f(x,y)上操作,定义在(x,y)的某个邻域上

  • T可以再一组图像上进行操作,例如噪声消除
  • 方形和矩形邻域是使用最多的,因为它们易实现,也可使用圆形区域

T最简单的情况是邻域大小 1*1。这时g只依赖于f在(x,y)处的值,即

s=T(r)

我们称之为灰度变换或两度变换,其中s与r分别为输出与输入的亮度值

# 幂律变换

比如伽马校正 对比度增强,伽马越小,越亮(对比度越高)如果图片亮度过高,也可以让伽马变大

至于对比度增强,也可以用分段线性函数

       matlab实现:

幂律变换函数实现的是imadjust(图像,[low_in,high_in],[low_out,high_out]) 其中low_in,high_in 映射到 low_out,high_out 不填Gamma就是最简单的直方图映射 [0.1] [1 0]就是反色 [] [] 2 gamma=2 前两个参数代表的是原图映射,但是gamma=2,即改成抛物线映射

G=imadjust(I,[0 1],[1 0]);
# 反色调,第三个参数缺省
imshow()


G = imadjust(I,[],[],2)
# 伽马=2,即 I(每个像素)^2
imshow()

# 对数变换

\begin{equation}

s = clog(1+r)

\end{equation}

  • 其中c是常数
  • 该变幻的形状类似于gamma曲线
  • 请注意,伽马曲线的形状是可变的,而对数函数的形状是固定的
  • 加一防止出现0的情况

对数变换常用于处理傅里叶变换后的频谱,使频谱的对比度更高,细节更清晰,傅里叶变换前通常需要对数变换

傅里叶谱在[0,1e6]的范围内并不少见,当显示在一个线性缩放到8位的显示器上时,高值占据了显示器的主导地位,从而丢失了低强度值的视觉细节频谱。通过计算log,动态范围可以减小到大约14,更易于显示管理 在执行对数转换时,通常希望将压缩后的值恢复到全部显示范围,对于8位,在Matlab中最简单的方法是使用:

gs = im2uint8(mat2gray(g))
1

使用mat2gray将值映射到[0,1]范围内,而im2uint8将值映射到[0,255]范围 使用对数变换的方式:

gs = im2uint8(mat2gray(log(1+double(f))));
imshow(gs)

# 椒盐噪声

       椒盐噪声的实现原理为，随机地将图像中的一定比例的像素值取极大或者极小：

       这里给出 Python 下的一种实现，可简单地转换为其他：

def salt_and_pepper_noise(x, v):
    # x 表示原始无噪图像，行数表示图像的个数，列数表示单幅图像的像素数。
    x_noise = x.copy()
    n_features = x.shape[1]
    mn = x.min()
    mx = x.max()
    for i, sample in enumerate(x):
        mask = np.random.randint(0, n_features, v)
        for m in mask:
            if np.random.rand() < .5:
                x[i][m] = mn
            else:
                x[i][m] = mx
    return x_noise
