# 变换

# 线性系统理论

       1. 线性系统

              许多图形处理系统都可以用一个线性系统作为模型:

       若系统满足:

  • aX->aY (a常数) -- 齐次性
  • x_1 ± x_2 -> y_1 ± y_2 -- 叠加性
  • a1X1 ± a2X2 -> a1Y1 ± a2Y2 (a1、a2常数) -- 线性性质

       即称为线性系统,否则即为非线性系统.

# 傅里叶变换

img = imread('men.bmp')
subplot(2,2,1);
imshow(img)
title('原图')
f = rgb2gray(img) % 对RGB或BMP图像进行灰度化
F=fft2(f); % 二维傅里叶变换
F1 = log(abs(F)+1); % 取模并进行缩放,将复数转换成实数(模)
subplot(2,2,2)
imshow(F1,[])
title('傅里叶变换频谱图')

Fs = fftshift(F) %将频谱图中零频率成分移至频谱图中心
S = log(abs(Fs)+1) %取模缩放
subplot(2,2,3);imshow(S,[]);title('频动后的频谱图');
fr = real(ifft2(ifftshift(Fs))); %频率反变换到空间域,取实部
ret = im2uint8(mat2gray(fr)) %更换图片类型
subplot(2,2,4);imshow(ret),title('逆傅里叶变换')
