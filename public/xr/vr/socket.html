# 粘包与分包

# 什么是粘包

如果单次传输数据过小,则多个数据可能会黏在一起发送到客户端,导致一个小型的Byte数组接收到的不止一组数据

# 什么是分包

如果一组数据过大,且服务端/客户端Byte缓冲数组不大,则可能会导致一组数据被拆成多个Byte数组

# 解决方案

在一组数据前加上一个字段,代表这组数据的长度

即

data_length + 数据

即前四个字节(一个int)代表数据长度

Message类提供:

Byte[] data=new byte[1024];
int startIndex=0;//我们存取了多少个字节的数据在数组里
以及缓存区剩余的空间
RemainSize=data.length-satrtIndex;

using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;
using System.Net.Sockets;
using System.Net;

namespace TCP服务器端
{
    class Program
    {
        static void Main(string[] args)
        {
            StartServerAsync();
            Console.ReadKey();
        }

        static void StartServerAsync()
        {
            Socket serverSocket = new Socket(AddressFamily.InterNetwork, SocketType.Stream, ProtocolType.Tcp);
            //本机Ip: 127.0.0.1
            //IpAddress xxx.xxx.xxx IpEndPoint xxx.xx.xx.xx:port
            //IPAddress ipAddress = new IPAddress(new byte[] { 127, 0, 0, 1 });
            IPAddress ipAddress = IPAddress.Parse("127.0.0.1");
            //ip - 端口
            IPEndPoint ipEndPoint = new IPEndPoint(ipAddress, 88);
            serverSocket.Bind(ipEndPoint);//绑定ip和端口号
            //同时等待处理队列
            serverSocket.Listen(0);
            //只能接受一个
            //Socket clientSocket = serverSocket.Accept();//接受一个客户端
            //接受多个客户端
            serverSocket.BeginAccept(AcceptCallBack,serverSocket);
        }

        static byte[] dataBuffer = new byte[1024];
        static void RecieveCallBack(IAsyncResult ar)
        {
            Socket clientSocket = null;
            try
            {
                clientSocket = ar.AsyncState as Socket;
                int count = clientSocket.EndReceive(ar);

                if(count == 0)
                {
                    clientSocket.Close();
                    return;
                }
                msg.AddCount(count);
                msg.ReadMessage();
                //string msgStr = Encoding.UTF8.GetString(dataBuffer, 0, count);
                //Console.WriteLine("从客户端接收到数据: " + msgStr);
                //循环接收
                clientSocket.BeginReceive(msg.Data, msg.StartIndex, msg.RemainSize, SocketFlags.None, RecieveCallBack, clientSocket);
            }
            catch (Exception e)
            {
                Console.WriteLine(e);
                if (clientSocket != null)
                {
                    clientSocket.Close();
                }
            }
        }

        static Message msg = new Message();
        static void AcceptCallBack(IAsyncResult ar)
        {
            Socket serverSocket = ar.AsyncState as Socket;
            Socket clientSocket = serverSocket.EndAccept(ar);

            //向客户端发送一条消息
            string msgStr = "Hello client!你好...";
            byte[] data = System.Text.Encoding.UTF8.GetBytes(msgStr);
            clientSocket.Send(data);

            clientSocket.BeginReceive(msg.Data, msg.StartIndex, msg.RemainSize, SocketFlags.None, RecieveCallBack, clientSocket);
            //接受多个客户端
            serverSocket.BeginAccept(AcceptCallBack, serverSocket);
        }

        void StartServerSync()
        {
            Socket serverSocket = new Socket(AddressFamily.InterNetwork, SocketType.Stream, ProtocolType.Tcp);
            //本机Ip: 127.0.0.1
            //IpAddress xxx.xxx.xxx IpEndPoint xxx.xx.xx.xx:port
            //IPAddress ipAddress = new IPAddress(new byte[] { 127, 0, 0, 1 });
            IPAddress ipAddress = IPAddress.Parse("127.0.0.1");
            //ip - 端口
            IPEndPoint ipEndPoint = new IPEndPoint(ipAddress, 88);
            serverSocket.Bind(ipEndPoint);//绑定ip和端口号
            //同时等待处理队列
            serverSocket.Listen(10);
            Socket clientSocket = serverSocket.Accept();//接受一个客户端

            //向客户端发送一条消息
            string msg = "Hello client!你好...";
            byte[] data = System.Text.Encoding.UTF8.GetBytes(msg);
            clientSocket.Send(data);

            //接受客户端的一条消息
            byte[] dataBuffer = new byte[1024];
            int count = clientSocket.Receive(dataBuffer);
            string msgReceive = System.Text.Encoding.UTF8.GetString(dataBuffer, 0, count);
            Console.WriteLine(msgReceive);
            clientSocket.Close();
            serverSocket.Close();
        }
    }
}

# Message.cs

using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;

namespace TCP服务器端
{
    class Message
    {
        private byte[] data=new byte[1024];
        private int startIndex = 0;

        public void AddCount(int count)
        {
            startIndex += count;
        }

        public byte[] Data
        {
            get { return data; }
        }
        public int StartIndex
        {
            get { return startIndex; }
        }
        public int RemainSize
        {
            get { return data.Length - startIndex; }
        }

        /// <summary>
        /// 解析数据/读取数据
        /// </summary>
        public void ReadMessage()
        {
            while (true)
            {
                //首次处理,必须存在长度
                if (startIndex <= 4) return;
                //从StartIndex位置读取Int32个字节
                int count = BitConverter.ToInt32(data, 0);
                //数据完整性检测
                if ((startIndex - 4) >= count)
                {
                    string s = Encoding.UTF8.GetString(data, 4, count);
                    Console.WriteLine("解析出来一条数据: " + s);
                    Array.Copy(data, count + 4, data, 0, startIndex - 4 - count);
                    startIndex -= (count + 4);
                }else
                {
                    break;
                }
            }
        }
    }
}

# 代码解读

serverSocket.Accept只能接收一个客户端的Socket
而ServerSocket.BeginAccept 则是开启一个异步的接收连接,当一个客户端的Socket接收到了以后,可以继续调用ServerSocket.BeginAccept来继续等待接受下一个客户端.

双端交互采用的发送数据的方式是这样的:

  1. 将数据转换为Byte数组
  2. 得到数据的长度(Byte长度)
  3. 将改长度转换为Byte数组(四字节)
  4. 连接1与3
public static byte[] GetBytes(string data)
{
    byte[] dataBytes = Encoding.UTF8.GetBytes(data);
    int dataLength = dataBytes.Length;
    byte[] lengthBytes = BitConverter.GetBytes(dataLength);
    byte[] newBytes = lengthBytes.Concat(dataBytes).ToArray();
    return newBytes;
}

所以每次发送的数据都是头部带有一个四字节整型代表长度的byte数组

至于如何读取数据,在Message.cs文件下,ReadMessage()

循环读取,每次读取前需要检查此次数据是否大于4字节,(因为第一个数据是长度)

成立的话,先读取前4个字节,得到一条数据的总长度(count).
然后将这条数据用子数组的方式解析出来(只解析一条数据).
在解析出来以后,再通过将剩余数据(如粘包时的数据)移至byte数组的头部,然后更新startIndex.

移至byte数组的头部的操作是通过

Array.Copy(byte数组,起始下标,目标Byte数组,起始下标,copy长度)

来实现的,这样一来,如果是分包问题,则下一次可以直接从缓冲区内读取到数据包.
如果是拆包问题,则可以和下一次接收到的包合并成一个完整的包.

Tip: 分包与粘包问题在服务器端与客户端都会出现,所以需要在客户端与服务端都对包进行相同的处理才可以.
