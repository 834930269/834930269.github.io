# Tesnorflow2 开篇

# 安装Tensorflow-gpu

前置条件: GPU算力必须在3.0以上

  1. 首先要安装cuda和cudnn
  2. cuda必须是10.0或者往下,建议就只用10.0.
  3. cudnn要7.4.6+，这之后的cudnn对GPU进行深度学习运算进行了优化
  4. Python - 3.7
  5. conda install tensorflow-gpu==2.0.0

# 安装失败解决方案

一般都是因为VS的插件安装错误

第一步: 控制面板中全删掉(包括Nvidia的显卡驱动,Cuda10安装中将驱动也安装了) 第二步: C:/ProgramData ProgramFile ProgramFile(x86)中包含和Nvidia的全删掉 第三部: 重启,安装(建议关掉360)

# 为什么使用GPU?

因为神经网络是使用矩阵进行运算,恰巧GPU也是使用矩阵进行运算(游戏中的渲染),通过GPU对神经网络的运算过程进行优化是再好不过的了.

# 检查是否装好

import tensorflow as tf
tf.test.is_gpu_available()
1
2

返回True即表示已经装好了

# 使用gpu进行运算

Tip: TF2只要检测到有gpu,直接就使用它了,不需要设置.

实在放不下心,就调用下面的代码

import os
# 选择编号为0的GPU
os.environ["CUDA_VISIBLE_DEVICES"] = "0"

其中0为要是用的gpu的编号

# 限制gpu使用大小

默认是完全使用gpu进行运算(如果找到的话)

import os
# 设置tf2-gpu按需增长
gpus = tf.config.experimental.list_physical_devices(device_type='GPU')
for gpu in gpus:
    tf.config.experimental.set_memory_growth(gpu, True)
