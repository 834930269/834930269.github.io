# 欧拉公式与初等群论

# 简要概述

       何谓初等群论:

Groups <-> Symmetry - 组 <-> 对称群

对称群,比如一个正方形可以沿着某个轴做任意的旋转 每次旋转都是关于该轴对称,即称为对称群

       同样的还有旋转群,立方体对称群,...        然后我们考虑等加群变换,比如一个长方形任意旋转360度,旋转10度,再反向旋转370度,旋转0度,这一系列的作用的最终结果相同,即等价,数学中的加法,乘法等概念也由此递推得来.

其中加法和乘法可以认作为作用叠加,什么叫作用叠加?加法群中的式子: 0-2= -2 的叠加效果如下图所示

       如图所示,加法群主要作用即平移,乘法群则为缩放 - 固定0点(原点)的缩放

# 向量群简述

       举个栗子:

       如上图所示,A = (4,3),我们知道,该等式又满足: A = (4,0) + (0,3),即两个向量相加(作用的叠加),则向量加法的定义即将这些作用相加,最终的结果即(4,3)，向右移动四位,向上移动三位...可以认为将原点右移四位,向上移动三位后原点的位置即A的位置,向量乘法可以类比.

# 复空间群

       何谓复空间?即纵坐标为i的倍数,横坐标为整数,至于i的定义,由后续的定义来表示.

复空间加法: 与向量的加法类似,只不过因为向量是单纯的数字,所以加法不能简单地用一个值来作为标量,而复数则可以加起来.

复数乘法群: 先做一个图示:

       我们想象一下,假设有一个钉子钉住了原点,那么我们抓住1这一点拉扯到任意地方(假设全激怒会等比例缩放),那当我们将1拉到1i时,坐标轴即旋转了90度

发现了么?

1 -> 1i 就等于旋转了90度

所以 复数乘法定义中: 1*i 即旋转90度

同理: 我们知道平常学习复数时, i*i = -1

为什么?

由图可知,当我们再乘上一个·i·时就再次旋转90度

而1i到-1的角度为90度.

即 i*i = -1

旋转30度的复数乘法群如下所示:

可以发现其角度是由实部与虚部共同作用产生的

插播一个理论: 同构

简单的同构(在这里)即代表加法与乘法的共同作用,并且在共同作用下保持结构不发生变化

如复指数群

这里将复指数群定义为 左侧加法群中平移i的倍数同样映射到右侧乘法群中单位圆上的点,单位元的弧即为加法群中的纵轴(i轴)

如 2^i 映射到右图约为0.693弧度值的旋转 5^i 约为1.609弧度的旋转 而 e的独特之处在于: e^i 即左侧加法群向上移动一个unit,右侧乘法群同步旋转一个弧度

且 e^(πi)刚好相当于右侧单位元旋转180度,即π弧度

这就是为什么 e^(πi) = -1

而这就是欧拉定理,e^(θi) 所等价的复空间的点就是旋转θ度的单位圆上的点 , 如 e^(π*i/2) = i

# 单位根

z = x + yi = r(cosθ+isinθ)=re^(iθ)

θ为辐角(旋转度数),r为模(长度)

       棣莫弗定理： z^2 = r^2e(i(θ1+θ2))

单位根:

z^n = 1 = r^n*e^(nθi) 的根

因为单位根的n次方为1,所以单位根的模r一定等于r=1 又因为单位根的n次方都落在点1处,所以 nθ=2kπ 联立上述两个方程组找到 z^n 的所有解 为: θ=2kπ/n r=1 即 z(n,k) = e^(i * 2kπ/n)

       将上述公式使用欧拉定理写成实部/虚部的形式:

z(n,k) = cos(2kπ/n) + i*sin(2kπ/n) k=(0,1,2,...,n-1) 详见下图:

# 多项式乘法

多项式乘法可以认为是离散的卷积,而fft在竞赛中常用来计算卷积或者多项式乘法. 朴素多项式乘法是O(n^2)复杂度 但是O(n^2)复杂度是基于系数表示情况下的 如果将其转换为点值表示,则可以使用FFT优化到O(nlgn)

# 多项式的点值表示

  • 把多项式放到平面直角坐标系里面，看成一个函数
  • 把n个不同的x代入，会得出n个不同的y，在坐标系内就是n个不同的点
  • 那么这n个点唯一确定该多项式，也就是有且仅有一个多项式满足∀k,f(x_k)=y_k

那么f(x)还可以用f(x)={(x0,f(x0)),(x1,f(x1)),(x2,f(x2)),...,(xn−1,f(xn−1))}来表示

这就是点值表示法,即n+1个点唯一确定一个n次函数

# 点值表示下的多项式乘法

用朴素方法来做复杂度是O(n^2) 用点值则只需要O(n)

故使用FFT来求系数多项式的点值形式,后直接将两式相乘就可以唯一确定相乘后的多项式,之后求值可以用插值,也可以逆变换回系数再进行求值(这个复杂度就高了)

# 离散傅里叶变换DFT (Discrete Fourier Transform)

DFT是为了将系数形式转换为点值形式

       既然多项式点值是随便取值,那就随便选择n个数带入就好了,那么代入哪些数呢?如果带入一个整数x,求取x的幂的复杂度依然是平方级别的,傅里叶选择带入单位根,至于为什么,可以看这个博客:

https://blog.csdn.net/enjoy_pascal/article/details/81478582

       离散傅里叶变换可以看做是一个将离散值从时域转换到频域的方式        离散傅里叶变换是对一个向量进行变换,可以将这个向量看做一个n-1次多项式,并在每个单位根处求值,得出向量y:

A(x) = lambda(i=0,n-1)ai*x^i

在w(n,0),w(n,1),...,w(n,n-1)处的值y0,y1,...,y(n-1)

yi = A(w(n,i)) = lambda(j=0,n-1)w(n,i×j)*aj

记为: y = DFT_{n}(a)

正常求值(多项式乘法)的话时间复杂度O(n^2)

# 快速傅里叶变换 FFT

O(nlgn) - 根据单位根的性质和分治思想来优化DFT

       第一步,将A(x)拆分为奇数项与偶数项,然后想办法将A(x,a)与A(x,a_奇数),A(x,a_偶数)项联立

    1. 将偶数项与奇数项同时替换参数x为x^2
    1. 奇数项乘上x

       根据单位根的消去引理和折半引理可以将上式简化:

上图最后两个式子为带入两个具体的值后的表达式,又因为:

\begin{equation} \omega {n}^{k+\frac{n}{2}} = e^{i\frac{2*(\frac{n}{2}+k)\pi }{n}} = e^{i\pi}e^{\frac{i2k\pi}{n}}=-1\omega{n}^{k} \end{equation}

       所以:

       问题转换成了在各个(奇偶)n/2次单位根上的值.这两个问题是原问题一半的子问题

至于为什么要求上面两个式子 可以发现这两个式子只有后面的一个符号不同 即我们只需要求A1(ω(k,n/2))和A0(ω(k,n/2))的值,我们就可以同时得到两个A的值

我们就只需要求两个 n/2 项的A,就可以得到两个n项的A

# BitReverse

对于FFT的优化,我们每次选择偶/奇数作为分治的目标,然后递归继续求奇偶,递归到最底层的到的顺序就可以用位逆序置换来一次性处理

如:

0 1 2 3 4 5 6 7

0 4 2 6 1 5 3 7

改写成位:

000 001 010 011 100 101 110 111

000 100 010 110 001 101 011 111

可以发现皆是位逆序.

位逆序置换计算方式的优化(找规律)

其中上一级子问题的解表示三位时,上一位代表两位二进制数的位逆序置换

对于式子的解释: i/2位置上的数右移一位,然后确定最高位的二进制数

只做一个优化

#include<stdc++.h>
using namespace std;

const double PI = acos(-1);

struct Complex {
	//实部,虚部
	double r, i;
	Complex() { r = 0, i = 0; }
	Complex(double real, double imag) :r(real), i(imag) {}
}F[maxn],G[maxn];
Complex operator + (Complex a, Complex b) {
	return Complex(a.r + b.r, a.i + b.i);
}
Complex operator - (Complex a, Complex b) {
	return Complex(a.r - b.r, a.i - b.i);
}
Complex operator * (Complex a, Complex b) { 
	return Complex(a.r * b.r - a.i * b.i, a.r * b.i + a.i * b.r); 
}

/*
**递归法实现
*/
void FFT(vector<Complex> a, int lim) {
	if (lim == 1) return;
	vector<Complex> a0(lim>>1), a1(lim>>1);
	//奇偶分类,写入数组,因为每次i+2,所以i>>1
	for (int i = 0; i < lim; i += 2) {
		a0[i >> 1] = a[i]; a1[i >> 1] = a[i + 1];
	}
	FFT(a0, lim >> 1);
	FFT(a1, lim >> 1);
	//合并前先定义两个变量
	//主n次单位根
	Complex wn = Complex(cos(2.0*PI / lim), sin(2.0*PI / lim));
	//旋转因子(w0,w1,w2...),w2=w1*w0,w3=w2*w1
	Complex w = Complex(1,0);
	//合并,因为选取的点的结果,每次可以计算出两个值,再根据旋转
	//每次可以计算出次数界个根
	for (int k = 0; k < (lim >> 1); ++k) {
		Complex t = w*a1[k];
		a[k] = a0[k] + t;
		a[k + (lim >> 1)] = a0[k] - t;
		w = w*wn;
	}
}


int main() {

	return 0;
}

# 非递归

#include<stdc++.h>
using namespace std;

const double PI = acos(-1);

struct Complex {
	//实部,虚部
	double r, i;
	Complex() { r = 0, i = 0; }
	Complex(double real, double imag) :r(real), i(imag) {}
}F[maxn],G[maxn];
Complex operator + (Complex a, Complex b) {
	return Complex(a.r + b.r, a.i + b.i);
}
Complex operator - (Complex a, Complex b) {
	return Complex(a.r - b.r, a.i - b.i);
}
Complex operator * (Complex a, Complex b) { 
	return Complex(a.r * b.r - a.i * b.i, a.r * b.i + a.i * b.r); 
}
const int maxn = 2000000+10;
int rev[maxn], len, lim = 1;
void FFT(Complex *a) {
	//根据预处理位逆序置换对其进行排序
	for (int i = 0; i < lim; ++i) if (i < rev[i]) swap(a[i], a[rev[i]]);
	for (int dep = 1; dep <= log2(lim); ++dep) {
		int m = 1 << dep;
		Complex wn = Complex(cos(2.0*PI / m), sin(2.0*PI / m));
		for (int k = 0; k < lim; ++k) {
			Complex w = Complex(1,0);
			for (int j = 0; j < m / 2; ++j) {
				Complex t = w*a[k + j + m / 2];
				Complex u = a[k + j];
				a[k + j] = u + t;
				a[k + j + m / 2] = u - t;
				w = w*wn;
			}
		}
	}
}

int main() {
	int n;
	scanf_s("%d", &n);
	while (lim <= n) lim <<= 1, len++;
	for (int i = 0; i < lim; ++i) rev[i] = (rev[i >> 1] >> 1) | ((i & 1) << (len - 1));
	return 0;
}

# FFT逆变换

即点值表示法转换成系数表示法

只需要在原代码中稍加修改即可同时求得逆变换和主变换

推导没仔细看.后期可以看看

用法: 多加一个参数opt

其中: opt=1代表离散傅里叶变换 opt=-1代表逆变换

#include<stdc++.h>
using namespace std;

const double PI = acos(-1);
const int maxn = 2000000 + 10;
struct Complex {
	//实部,虚部
	double r, i;
	Complex() { r = 0, i = 0; }
	Complex(double real, double imag) :r(real), i(imag) {}
}F[maxn],G[maxn];
Complex operator + (Complex a, Complex b) {
	return Complex(a.r + b.r, a.i + b.i);
}
Complex operator - (Complex a, Complex b) {
	return Complex(a.r - b.r, a.i - b.i);
}
Complex operator * (Complex a, Complex b) { 
	return Complex(a.r * b.r - a.i * b.i, a.r * b.i + a.i * b.r); 
}

int rev[maxn], len, lim = 1;
void FFT(Complex *a,int opt) {
	//根据预处理位逆序置换对其进行排序
	for (int i = 0; i < lim; ++i) if (i < rev[i]) swap(a[i], a[rev[i]]);
	for (int dep = 1; dep <= log2(lim); ++dep) {
		int m = 1 << dep;
		Complex wn = Complex(cos(2.0*PI / m), opt * sin(2.0*PI / m));
		for (int k = 0; k < lim; k+=m) {
			Complex w = Complex(1,0);
			for (int j = 0; j < m / 2; ++j) {
				Complex t = w*a[k + j + m / 2];
				Complex u = a[k + j];
				a[k + j] = u + t;
				a[k + j + m / 2] = u - t;
				w = w*wn;
			}
		}
	}
	if (opt == -1) for (int i = 0; i < lim; ++i) a[i].r /= lim;
}

int main() {
	int n;
	scanf_s("%d", &n);
	while (lim <= n) lim <<= 1, len++;
	for (int i = 0; i < lim; ++i) rev[i] = (rev[i >> 1] >> 1) | ((i & 1) << (len - 1));
	return 0;
}

# 示例 - FFT求两多项式乘积 - 即两向量卷积

  • 先将两个多项式变成点值表达形式,然后O(n)进行点乘运算 又称为卷积
  • 再在O(nlgn)时间内逆变换
  • 注意,运算时必须扩大到2n,因为卷积后的系数个数是2n个,这样才不会出错
  • 且两个向量的长度应该统一到同一2的幂次,且输入时的长度应该扩大到一倍

# 代码

#include<stdc++.h>
using namespace std;

const double PI = acos(-1);
const int maxn = 2000000 + 10;
struct Complex {
	//实部,虚部
	double r, i;
	Complex() { r = 0, i = 0; }
	Complex(double real, double imag) :r(real), i(imag) {}
}F[maxn],G[maxn];
Complex operator + (Complex a, Complex b) {
	return Complex(a.r + b.r, a.i + b.i);
}
Complex operator - (Complex a, Complex b) {
	return Complex(a.r - b.r, a.i - b.i);
}
Complex operator * (Complex a, Complex b) { 
	return Complex(a.r * b.r - a.i * b.i, a.r * b.i + a.i * b.r); 
}

int rev[maxn], len, lim = 1;
void FFT(Complex *a,int opt) {
	//根据预处理位逆序置换对其进行排序
	for (int i = 0; i < lim; ++i) if (i < rev[i]) swap(a[i], a[rev[i]]);
	for (int dep = 1; dep <= log2(lim); ++dep) {
		int m = 1 << dep;
		Complex wn = Complex(cos(2.0*PI / m), opt * sin(2.0*PI / m));
		for (int k = 0; k < lim; k+=m) {
			Complex w = Complex(1,0);
			for (int j = 0; j < m / 2; ++j) {
				Complex t = w*a[k + j + m / 2];
				Complex u = a[k + j];
				a[k + j] = u + t;
				a[k + j + m / 2] = u - t;
				w = w*wn;
			}
		}
	}
	if (opt == -1) for (int i = 0; i < lim; ++i) a[i].r /= lim;
}

int main() {
	int n,m;
	//两个多项式次数
	scanf_s("%d %d", &n,&m);
	for (int i = 0; i <= n; ++i)scanf_s("%lf", &F[i].r);
	for (int i = 0; i <= m; ++i)scanf_s("%lf", &G[i].r);
	len = 0;
	while (lim <= n+m) lim <<= 1, len++;
	for (int i = 0; i < lim; ++i) rev[i] = (rev[i >> 1] >> 1) | ((i & 1) << (len - 1));

	FFT(F, 1); FFT(G, 1);
	//FFT后点值计算
	for (int i = 0; i <= lim; ++i) F[i] = F[i] * G[i];
	//逆变换
	FFT(F, -1);

	for (int i = 0; i <= n + m; ++i) {
		printf("%d ", static_cast<int>(F[i].r + 0.5));
	}

	return 0;
}
